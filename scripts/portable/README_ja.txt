 Wildlife Detector v2.2.0 (ポータブル版)
 野生生物検出デスクトップアプリケーション
 Powered by Google SpeciesNet
============================================================

■ 使い方

  1) この zip ファイルを任意のフォルダに解凍してください。
     （例: C:\Tools\WildlifeDetector\ など）
     ※ 日本語やスペースを含むパスでも動作しますが、可能なら
       半角英数だけのフォルダを推奨します。

  2) 解凍したフォルダ内の「WildlifeDetector.bat」をダブルクリック
     してください。

  3) 初回起動時は SpeciesNet の学習済みモデル（約500MB）を
     自動的にダウンロードします。
       - ネットワーク接続が必要です（完了まで数分かかります）
       - 保存先: %USERPROFILE%\.cache\
     2回目以降の起動ではダウンロードは不要で、オフラインでも
     動作します。


■ 動作環境

  - Windows 10 / 11 (64bit)
  - 空き容量: 5GB 以上
  - メモリ: 8GB 以上推奨
  - 初回のみネットワーク接続


■ 使い方（アプリ画面）

  1) 「入力・設定」タブで画像フォルダを選択
  2) 出力フォルダを指定
  3) 「検出処理開始」をクリック
  4) 「結果」タブで検出結果を確認
     - 100件ずつページング表示されます
     - 「日付×種別ピボット出力」で集計CSVを生成できます


■ 出力ファイル

  検出結果CSV / 種リスト / サマリー / 日付×種別ピボットCSV はすべて
  UTF-8 BOM 付きで保存され、Excel でダブルクリックして開けます。

  - 種名は「学名 (英名 / 和名)」併記で出力されます
    例: Corvus macrorhynchos (large-billed crow / ハシブトガラス)
  - 検出結果CSVには common_name (英名) と japanese_name (和名) が
    別列で含まれます。和名が登録されていない種は japanese_name 列が
    空欄になります。
  - カテゴリ列は「鳥類 / 哺乳類 / 爬虫類 / 両生類 / 魚類 / 昆虫類 /
    未検出」と日本語で表記されます。
  - 振り分けフォルダ名は「和名_学名」形式（例: ハシブトガラス_Corvus_macrorhynchos）で
    作成されます。和名未登録の場合は英名 / 学名のみで作成されます。


■ トラブル時

  1) WildlifeDetector-debug.bat を実行してください。
     黒いコンソール画面にエラーメッセージが表示されます。

  2) ログファイルの場所:
       %USERPROFILE%\WildlifeDetector\logs\wildlife_detector.log

  3) 「Qt platform plugin "windows" が見つからない」系のエラーが
     出る場合は、以下を Microsoft からインストールしてください:
       "Microsoft Visual C++ 2015-2022 再頒布可能パッケージ (x64)"

  4) 解決しない場合は上記ログファイルを添えて配布元までご連絡
     ください。


■ アンインストール

  解凍したフォルダをまるごと削除してください（レジストリは
  使用していません）。
  学習済みモデルのキャッシュも消す場合は下記フォルダを削除:
    %USERPROFILE%\.cache\
